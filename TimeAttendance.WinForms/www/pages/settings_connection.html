Cài đặt kết nối (nhập IP)

Dùng khi quán đổi mạng / đổi IP. Nhập IP + Port mới và bấm Lưu. QR sẽ dùng địa chỉ mới (không cần build lại exe).

wifi

Địa chỉ API

Đây là IP LAN của máy đang chạy API (port 5000).

QR sẽ dùng preview này để điện thoại mở trang chấm công.

Gợi ý

  • Nếu điện thoại không quét được QR, hãy kiểm tra máy chạy API và WiFi cùng mạng LAN.
  • Nếu đổi router / đổi WiFi, IP LAN có thể đổi — chỉ cần cập nhật lại ở đây.
